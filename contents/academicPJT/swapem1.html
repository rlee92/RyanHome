Swap'em for iOS

Swap'em simplifies the process of exchaningc contact information small

Demonstration Video


Built with: JavaScript, React-Native, Xcode

  • A mobile app that simplifies the contact exchanging process with nearby users
  • Each profiles are customizable and users can select which information they would like share
  • Received contacts can be remained in app or imported into native contact app

Developed front-end features:
  • Contributed profile creation page and profile page
  • Contributed to add a map that shows where the users exchanged each contact information
  • Integrated with Facebook API to display a user's Facebook URL on profile page in app
